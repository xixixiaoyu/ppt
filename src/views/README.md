# Home.vue 优化说明

## 概述

本次优化对 `src/views/Home.vue` 进行了全面重构，将原本简单的演示文稿列表页面升级为功能丰富、用户体验优秀的现代化界面。特别注重空间利用效率，提供了紧凑而功能丰富的布局。

## 主要改进

### 1. 数据配置化

- **文件**: `src/config/presentations.ts`
- **功能**: 将硬编码的数据抽离到配置文件，支持更丰富的元数据
- **新增字段**: 难度、时长、更新时间、标签、描述等

### 2. 主题系统

- **文件**: `src/config/themes.ts`, `src/components/ThemeToggle.vue`
- **功能**: 支持 5 种主题切换（深色、浅色、蓝色、紫色、绿色）
- **特性**: 自动保存用户偏好，支持系统主题检测

### 3. 搜索和筛选功能

- **文件**: `src/components/SearchAndFilter.vue`
- **功能**:
  - 全文搜索（标题、描述、标签）
  - 分类筛选
  - 难度筛选
  - 标签多选筛选
  - 多种排序方式
  - 筛选状态指示器

### 4. 组件化架构

- **文件**: `src/components/PresentationList.vue`
- **功能**: 将演示文稿列表拆分为独立组件
- **特性**:
  - 分类展开/折叠动画
  - 难度标识
  - 元信息展示
  - 标签系统
  - 空状态处理

### 5. 键盘导航

- **文件**: `src/components/KeyboardShortcuts.vue`, `src/utils/composables.ts`
- **快捷键**:
  - `/`: 聚焦搜索框
  - `f`: 打开/关闭筛选器
  - `e`: 展开所有分类
  - `c`: 折叠所有分类
  - `t`: 切换主题
  - `Esc`: 关闭弹窗/清除搜索

### 6. 响应式设计和空间优化

- **功能**: 完全响应式布局，支持移动端、平板和桌面
- **特性**:
  - 移动端优化的触控体验
  - 自适应布局
  - 优化的字体和间距
  - 紧凑型设计，最大化空间利用
  - 智能视图切换（移动端自动使用列表视图）

### 7. 动画和过渡效果

- **功能**: 丰富的微交互动画
- **包含**:
  - 分类展开/折叠动画
  - 筛选面板滑入/滑出
  - 悬停效果
  - 主题切换动画
  - 加载状态动画

### 8. 性能优化和空间利用

- **功能**: 多项性能优化措施和空间利用优化
- **包含**:
  - 组件懒加载
  - 计算属性缓存
  - 事件监听器清理
  - 虚拟滚动准备
  - 紧凑型网格布局，提高信息密度
  - 智能内容折叠和展开

## 技术栈

- **Vue 3**: Composition API
- **TypeScript**: 类型安全
- **Tailwind CSS**: 样式系统
- **Heroicons**: 图标库
- **组合式函数**: 逻辑复用

## 文件结构

```
src/
├── config/
│   ├── presentations.ts    # 演示文稿数据配置
│   └── themes.ts         # 主题配置
├── components/
│   ├── SearchAndFilter.vue      # 搜索筛选组件（支持紧凑模式）
│   ├── PresentationGrid.vue     # 演示文稿网格/列表组件
│   ├── PresentationList.vue     # 原演示文稿列表组件（已弃用）

│   ├── ViewToggle.vue         # 视图模式切换组件
│   └── KeyboardShortcuts.vue  # 键盘快捷键组件
├── utils/
│   └── composables.ts     # 组合式函数
└── views/
    └── Home.vue          # 主页面（已优化）
```

## 使用说明

1. **搜索**: 在搜索框中输入关键词，支持标题、描述和标签搜索
2. **筛选**: 点击筛选按钮打开筛选面板，可按分类、难度、标签筛选
3. **排序**: 在筛选面板中选择排序方式
4. **主题**: 点击右上角主题按钮切换主题
5. **键盘**: 使用键盘快捷键快速操作
6. **分类**: 点击分类标题展开/折叠演示文稿列表

## 扩展性

- **新增演示文稿**: 在 `src/config/presentations.ts` 中添加数据
- **新增主题**: 在 `src/config/themes.ts` 中配置主题
- **新增筛选**: 扩展 `useSearchAndFilter` 组合式函数
- **新增快捷键**: 在 `useKeyboardNavigation` 中添加回调
- **自定义布局**: 修改 `PresentationGrid.vue` 中的网格列数
- **紧凑模式调整**: 在 `SearchAndFilter.vue` 中调整紧凑程度

## 兼容性

- 现代浏览器（Chrome 88+, Firefox 85+, Safari 14+）
- 移动端和桌面端
- 支持触摸和鼠标交互
